测试并行压缩算法(pigz/pbzip2)

Date Tue 25 June 2013 By renewjoy Category 玩意. Tags gz bz2 bzip2 pigz pbzip2

测试环境

  • 机器cpu：4*2.4
  • 系统： Ubuntu x64 12.04
  • 测试文件：/data/logdev/file_20121103.0000-2400.log

安装测试环境

安装pbzip2

  • sudo apt-get install pbzip2 安装的为1.1.6版本
sudo apt-get install libbz2-dev
tar xvf pbzip2-1.1.8.tar.gz
cd pbzip2-1.1.8
make
sudo make install
$ pbzip2 --version
Parallel BZIP2 v1.1.8 - by: Jeff Gilchrist `http://compression.ca <http://compression.ca>`_
[Jun. 10, 2012]               (uses libbzip2 by Julian Seward)
Major contributions: Yavor Nikolov <nikolov.javor+pbzip2@gmail.com>

安装pigz

sudo apt-get install libzip-dev
wget http://zlib.net/pigz/pigz-2.3.tar.gz
make(编译不成功）
sudo apt-get install pigz
$ pigz --version
pigz 2.1.6

压缩比率对比

压缩类型 大小 比率 占未压缩的比例
未压缩 6.3G
gz 876M 14% 大约：1/7
pigz 878M 14% 大约：1/7
bzip2 513M 8% 大约：1/12
pbzip2 514M 8% 大约：1/12

压缩时间对比

压缩算法 压缩时间 解压时间
gz 3分25秒 1分13秒
pigz 1分12秒 1分19秒
bzip2 20分19秒 4分2秒
pbzip2 5分27秒 1分38秒
